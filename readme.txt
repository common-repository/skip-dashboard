== Description ==

This lightweight plugin allows WP admin user to skip default wordpress dashboard after login and jump straight to specific page.

The plugin is useful to jump straight to frequently visited admin page right after wp login, skipping default dashboard page.

== Installation ==

1. Install the plugin from within the Dashboard or upload the directory `skip-dashboard` and all its contents to the `/wp-content/plugins/` directory of your website

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Set your preferred URL on Settings->Skip Dashboard.

4. That's all, Now when user logged in into wp-admin, they will be automatically redirected to the new URL (set in step 3) skipping default wordpress dashboard.

== Frequently Asked Questions ==

= Does it overwrite wordpress user permission for admin section?  =

This plugin doesn't change any admin user permission, it only redirect admin user to new URL instead of default WordPress dashboard.

= Are all the users will be redirected?  =

This plugin will only redirect users with <strong>Administrator, Editor and Author</strong> privilege.
